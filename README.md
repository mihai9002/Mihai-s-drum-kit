# Mihai-s-drum-kit:
a macropad drum kit with a digital encoder, so i can switch between presets, and a display to see the current preset
# Components:
- 128x32 OLED Display
- EC11 Rotary encoder for switching between presets
- 7 Switches, the "drums"
